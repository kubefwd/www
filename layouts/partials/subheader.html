kubefwd is used to forward Kubernetes services running in a remote cluster for the purpose of local development. kubefwd, pronounced Kube Forward, is a single binary, command line application written in Go.

kubefwd does not require any modifications to clusters it connects to. If you currently use kubectl, you have met the requirements for kubefwd.
